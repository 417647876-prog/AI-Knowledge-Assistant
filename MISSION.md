# Mission: 从 C# 后端开发转向 AI 应用开发

## Why

以现有 C#/.NET 后端经验为基础，独立完成一个可运行、可测试、可讲解的企业知识库 RAG 项目，并具备求职或实际项目开发所需的 Python AI 工程能力。

## Success looks like

- 能解释文档解析、切片、Embedding、向量检索和引用生成的完整链路。
- 能使用 FastAPI、SQLAlchemy、PostgreSQL 和 pgvector 编写可测试的 RAG 后端。
- 能把 Python 项目结构与熟悉的 ASP.NET Core 概念对应起来。

## Constraints

- 代码小步修改并配套中文解释。
- 优先完成可运行的后端闭环，不提前增加前端、权限和 Agent。

## Out of scope

- 当前阶段不学习模型训练、微调、Kubernetes 和复杂多 Agent 系统。
