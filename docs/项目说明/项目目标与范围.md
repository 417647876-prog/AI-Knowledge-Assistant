# 项目目标与范围

## 项目定位

以现有 C#/.NET 后端经验为基础，独立完成一个可运行、可测试、可讲解的企业知识库 RAG 项目，并形成可用于求职演示的 Python AI 工程作品。

## 已完成范围

- 文档上传、解析、清洗、切片、向量入库和可追溯引用。
- 本地语义检索、关键词混合检索、重排序、问题改写和拒答边界。
- Vue 工作台、账号认证、角色权限、知识库隔离和流式多轮问答。
- 固定评估集、质量门、综合验收报告和自动化测试。
- PostgreSQL 持久化任务、独立 Worker、租约恢复、严格所属人隔离和临时只读支持授权。
- 回收站、服务端会话与用量、额度与限速、日志/指标/审计、脱敏运营接口和完整 Docker Compose。
- 响应式手机 H5、桌面管理员运营界面、前端路由拆包，以及 PostgreSQL/uploads 的显式备份恢复。
- Tailscale Funnel 显式启停代码和手机验收手册；真实公网 HTTPS 与手机移动网络证据仍待首次登录授权后补齐。

## 当前目标

- 完成阶段 5 的用户硬门：Tailscale 首次登录/授权，以及真实手机关闭 Wi-Fi 后的公网 HTTPS 十步验收。
- 保持项目可在本机稳定复现，并能在面试中解释架构选择、质量证据和安全边界。

## 学习定位

- 重点掌握 FastAPI、Pydantic、SQLAlchemy、PostgreSQL、pgvector 与 RAG 的工程实践。
- 使用 ASP.NET Core Controller、DTO、EF Core、DI 等已有经验理解 Python 对应概念。
- 每个阶段先形成可验证闭环，再继续扩展功能。

## 当前非目标

- 模型训练或微调、复杂多 Agent 编排、Kubernetes 和生产级高可用。
- 公开注册、支付、组织协作、正式云服务器和微信小程序。
